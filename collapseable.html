SyncHole

The SyncHole is a small set of web pages with deliberate issues associated with Synchronisation,

Manually interacting with the pages should be fairly simple, but there are some deliberate bugs in there.

Just for educational value.

They are unrelated to the actual functionality for automating.

Some of the functionality you could automate with implicitWaits, some with Thread.sleep if you really wanted to, but a lot of the timings are randomly generated.

Ultimately, we want you to be using explicit waits.
